Django

Django est un cadre de développement web source ouverte en Python. Il a pour but de rendre le développement web 2.0 simple et rapide. Pour cette raison, le projet a pour slogan « Le framework web pour les perfectionnistes sous pression ». Développé en 2003 pour le journal local de Lawrence (Kansas), Django a été publié sous licence BSD à partir de juillet 2005.

Depuis juin 2008, la Django Software Foundation s'occupe du développement et de la promotion du cadre. En plus de cette promotion régulière, des conférences entre développeurs et utilisateurs de Django sont organisées deux fois par an depuis 2008. Nommées DjangoCon, une se déroule en Europe et l'autre aux États-Unis.

Plusieurs sites grand public sont désormais fondés sur Django, dont Pinterest4 et Instagram5 au moins en 2011 ou encore Mozilla.

Les templates en mode statique

C'est pourtant simple :

  • tu fais un fichier html template que tu mets dans le répertoire templates de ton app
  • tu configures une route dans le fichier urls.py de ton app qui pointe sur une vue
  • dans cette vue, tu fais appel à la fonction render()

That's all folks!

Templates

Ca n'est pas compliqué :

  • tu fais un fichier html template de base (par ex base.html) que tu mets dans le répertoire templates de ton app ou dans un repertoire global definit dans le parametre DIRS de TEMPLATES
  • tu y inclus des blocs du style

    {% block mon_bloc %}

    du html...

    {% endblock %}

  • tu inclus dans tes templates un {% extends 'base.html' %}
  • tu configures une route dans le fichier urls.py de ton app qui pointe sur une vue
  • dans cette vue, tu fais appel à la fonction render() en lui passant un template en parametre...
En plus de cela, tu peux utiliser des balises pour faire :
  • l'inclusion de variables avec {{ ma_variable }}
  • des boucles avec {% for xx in yy %}
  • des conditions avec {% if mon_parametre == 2 %}
  • de l'inclusion d'un autre template avec {% include 'autre/template.html' %}

Django c'est drôlement chouettte !
